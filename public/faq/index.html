Frequently Asked Questions

How much does JXOS cost?

Nothing, JXOS is provided for free.

Any warranty or guarantee provided?

JXOS is provided for you as is, without any guarante or warranty.

Why LiveISO mines to Developer wallets on start?

The automine is for testing the operating system.

Supported Graphic Cards

It is tested against : RX580-8GB, GTX750TI, GTX1060-3GB, GTX1070 and GTX1070TI.

Is mixing AMD and Nvidia GPU in a single box possible?

It is possible, but it is untested feature so your milage may vary.

Is running both CPU and GPU miner possible?

Yes, it is possible.

Is wireless connection supported?

No, wireless support must be installed separately.

Do you use static ip for the miner box?

By default it will use DHCP to detect the miner box IP address, but it is recommended to change it to static ip address for faster booting process.

Is this open source?

Yes, it is open sourced whenever possible.

What miners does this os support?

It supports XMRig (CPU, Nvidia and AMD), CCMiner, Claymore (ETH, ZEC and XMR), ETHMiner, EWBF, SGMiner, CastXMR, Avermore, T-Rex, Cryptodredge, TeamRedMiner and Wildrig.

What is the Linux distro this OS based on?

This OS is based on Ubuntu Linux distro

What is the default user and password?

The default preconfigured username is jxminer with password of jxminer
